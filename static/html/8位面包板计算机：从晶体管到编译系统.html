8位面包板计算机：从晶体管到编译系统

2022年8月份购置了一份8位计算机面包板组装套件，之后一直想把它组装完成，但是各种原因，都没有着手。正好这两年打算自考本科计算机，可以学完理论之后，重拾这个组装，自己动手从最底层开始一步步实现一台计算机。

C语言

历史

1972年，丹尼斯·里奇与肯·汤普逊为了更好地移植和开发UNIX操作系统，以B语言为基础，在贝尔实验室设计、开发出C语言。

1983年，美国国家标准协会组成了一个委员会，X3J11，为了创立C的一套标准。经过漫长而艰苦的过程，该标准于1989年完成，并在作为ANSI X3.159-1989 "Programming Language C"正式生效。

JavaScript

历史

1995年，Netscape公司为了在浏览器领域胜过微软，推出了一种语言，可以在客户端就能完成表单的简单验证。这种语言后来被命名为JavaScript，与此同时，微软也推出自己的类似脚本语言JScript。1997年，以JavaScript1.1为蓝本的建议被提交给ECMA，由此ECMAScript诞生，这个标准成各种JavaScript实现的基础。

2009年，ECMAScript第五版，即规范ECMA-262第五版发布。相比于1997年的第一版，这一版本增加了正则表达式、JSON对象等特性的支持。

基本概念

ECMAScript只规定了JavaScript这门语言的最基本的内容，如语法、关键字、保留字、变量、数据类型、操作符、语句、函数，而对于与Web浏览器联系紧密的DOM和BOM没有涉及。因此众多宿主环境，如Node、Adobe Flash、Autojs等在实现ECMCScript标准的同时，可以有自己的扩展。

总的来说，JavaScript是一种专为网页交互而设计的脚本语言，主要由两个部分组成。一个是ECMAScript，有ECMA-262规范定义，提供核心语言功能；另一个则是扩展功能，用来提供与宿主环境交互的方法和接口。

Python

历史

2000年，Python 2.0发布，引入了列表推导式。

2008年，Python 3.0发布，这一版本对语言做了较大修改而不能完全向后兼容。
